Computer Science Department Course Info API Demo

This program allows users to get info about the courses in the computer science department here at St. John's by utilizing an API. This restful API acts as a middle man between the back-end database which holds all of the data, so that it can be easily transmitted to a program like this one. With different functions to return specific things based on what the user is looking for.

This type of service could be very beneficial as it allows the info about our programs to be used more broadly on third party applications, and can help build even further the graduate as well as undergraduate programs.
